Common Questions

Do I need to be tech-savvy to use this?

No. If you can send a text message, you can use this system. We handle all the technical setup — you just talk to your AI like you'd text a colleague.

What CRM do I need?

We currently integrate with Follow Up Boss. If you use a different CRM, ask us — most major platforms have APIs we can connect to.

Does the AI send messages without me approving them?

Only if you want it to. We can configure it either way — full automation, or draft-and-approve. Most agents start with approve-first until they trust the system.

Will the AI sound like me?

We build a voice profile based on your actual messages and communication style. The more you use it, the more it adapts. It's not perfect on day one — but it gets there.

What's the monthly fee for?

AI tools update constantly. The monthly fee covers keeping your system running, updating integrations when platforms change, and being available when something needs adjusting.

Can I start with just one pillar?

Yes. Pillar 01 (the AI setup) is the foundation — everything else adds on. You can start with just the core and expand when you're ready.
