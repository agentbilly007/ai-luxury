Common Questions

Do I need to be tech-savvy?

No. If you can send a text message, you can use this. I handle all the technical setup. You just talk to your AI like you'd text a colleague.

What CRM do I need?

I currently integrate with Follow Up Boss. If you use a different CRM, reach out — most major platforms have APIs I can connect to.

Will the AI send messages without my approval?

Only if you want it to. Most agents start with draft-and-approve mode until they trust the system. We configure it however you're comfortable.

What's the difference between the three plans?

The $100 Guided Call is a Zoom session — you install it yourself, I walk you through every step. The $200 Full Setup means I handle the entire installation for you. Done-For-You at $1,000/month means I manage your AI operation ongoing — automations, follow-up, content, all of it. All three include access to the community.

Will the AI sound like me?

We build a voice profile based on how you actually communicate. The more you use it, the better it gets. It's not perfect on day one — but it gets there fast.
